# ALPSEREN — Instructions pour Claude Code

## Identité du projet
Site web de la marque ALPSEREN — conciergerie de luxe alpine.
Stack : Next.js 16 + Tailwind CSS v4 + TypeScript

## Règles absolues — ne jamais enfreindre
- Couleurs UNIQUEMENT : #C29B6D (beige sable), #2D2926 (anthracite), #F9F9F9 (fond)
- Fonts UNIQUEMENT : Cinzel (titres) + Montserrat Light 300 (corps)
- ALPSEREN toujours en majuscules
- Fond toujours #F9F9F9, jamais #ffffff
- Texte toujours #2D2926, jamais #000000
- Zéro bouton coloré — uniquement liens texte avec underline
- Zéro emoji, zéro icône décorative
- Espaces généreux (padding minimum py-16 par section)
- Animations discrètes max 1s, easing cubic-bezier(0.16, 1, 0.3, 1)

## Logo
- Fichier : /public/logo.png
- Baseline : PRIVATE ESTATE & LIFESTYLE
- Alt text : ALPSEREN — Private Estate & Lifestyle

## Contenu bilingue
- Langue par défaut : FR
- Switcher FR · EN dans le header (texte uniquement, pas de drapeaux)

## Librairies installées
- gsap (animations scroll)
- lenis (smooth scroll)
